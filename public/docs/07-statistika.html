📊 Στατιστικά & Αναφορές

Πώς να παρακολουθείς τα έσοδα, τα έξοδα και τα κέρδη σου

Τι είναι η σελίδα «Στατιστικά»;

Στη σελίδα Στατιστικά βλέπεις αναλυτικά γραφήματα και αριθμούς για τη δουλειά σου. Εδώ μπορείς να:

  • ✅ Δεις πόσα έσοδα έχεις κάνει συνολικά
  • ✅ Υπολογίσεις τα καθαρά κέρδη σου
  • ✅ Παρακολουθήσεις τα έσοδα ανά μήνα
  • ✅ Δεις πόσες εργασίες έχεις κάνει και τι κατάσταση έχουν
  • ✅ Δεις ποια υλικά χρησιμοποιείς περισσότερο
  • ✅ Φιλτράρεις τα στατιστικά ανά έτος

💡 Πολύ χρήσιμο!

Με τα στατιστικά μπορείς να δεις αν βγάζεις λεφτά, ποιοι μήνες είναι καλύτεροι, και να πάρεις καλύτερες αποφάσεις για τη δουλειά σου!

Κάρτες Συνόλων

Στην κορυφή της σελίδας υπάρχουν 4 κάρτες που δείχνουν τα βασικά νούμερα:

💵 Συνολικά Έσοδα

Πόσα χρήματα έχεις βγάλει συνολικά από όλες τις ολοκληρωμένες εργασίες.

💰 Καθαρά Κέρδη

Έσοδα μείον έξοδα = Πόσα σου έμειναν καθαρά!

💼 Σύνολο Εργασιών

Πόσες δουλειές έχεις κάνει συνολικά.

✅ Εξοφλημένες

Πόσες εργασίες έχουν πληρωθεί.

Φίλτρο Έτους

Πάνω δεξιά υπάρχει ένα dropdown με έτη. Επίλεξε το έτος που θέλεις να δεις και όλα τα γραφήματα θα ενημερωθούν αυτόματα!

📅 Παράδειγμα

Αν επιλέξεις «2024», θα δεις μόνο τα στατιστικά για τις εργασίες που έγιναν το 2024. Αν επιλέξεις «Όλα τα έτη», θα δεις συνολικά στατιστικά από όλες τις εργασίες!

Γραφήματα

Η σελίδα έχει 4 κύρια γραφήματα:

1. 📅 Έσοδα ανά Μήνα

Τι δείχνει: Ένα γράφημα με μπάρες που δείχνει πόσα έσοδα έχεις κάνει κάθε μήνα.

Γιατί είναι χρήσιμο: Βλέπεις ποιοι μήνες είναι καλοί (πολλά έσοδα) και ποιοι είναι πιο αδύναμοι. Έτσι μπορείς να προγραμματίσεις καλύτερα!

✅ Παράδειγμα

Αν βλέπεις ότι τον Αύγουστο έχεις λίγα έσοδα, μπορεί να πάρεις διακοπές ή να κάνεις καμπάνια για να βρεις νέους πελάτες εκείνον τον μήνα!

2. 🎨 Κατανομή Εργασιών

Τι δείχνει: Ένα γράφημα πίτας (pie chart) που δείχνει πόσες εργασίες έχεις ανά τύπο (Εσωτερικοί χώροι, Εξωτερικοί χώροι, Κάγκελα κ.λπ.).

Γιατί είναι χρήσιμο: Βλέπεις τι είδους δουλειές κάνεις περισσότερο. Αν κάνεις κυρίως εσωτερικούς χώρους, μπορεί να αγοράσεις περισσότερα υλικά για εσωτερικά!

3. ✅ Κατάσταση Εργασιών

Τι δείχνει: Ένα γράφημα πίτας που δείχνει πόσες εργασίες είναι σε κάθε κατάσταση (Ολοκληρωμένες, Σε εξέλιξη, Εξοφλημένες κ.λπ.).

Γιατί είναι χρήσιμο: Βλέπεις πόσες δουλειές έχεις στον αέρα και πόσες έχουν τελειώσει. Αν έχεις πολλές «Σε αναμονή», μπορεί να πρέπει να τις διαχειριστείς!

4. 📦 Top 10 Υλικά

Τι δείχνει: Ένα γράφημα με μπάρες που δείχνει ποια χρώματα/υλικά χρησιμοποιείς περισσότερο.

Γιατί είναι χρήσιμο: Αν βλέπεις ότι χρησιμοποιείς πολύ ένα συγκεκριμένο χρώμα (π.χ. "Λευκό Ματ"), μπορείς να το έχεις πάντα σε στοκ!

💡 Πώς υπολογίζεται;

Το πρόγραμμα μετράει πόσες φορές έχεις χρησιμοποιήσει κάθε μάρκα χρώματος στις εργασίες σου και σου δείχνει τα 10 πιο δημοφιλή!

Πώς υπολογίζονται τα έσοδα και τα έξοδα;

Έσοδα

Τα έσοδα υπολογίζονται από τις εργασίες που είναι:

  • «Ολοκληρώθηκε» (τελείωσαν αλλά δεν έχουν πληρωθεί ακόμα)
  • «Εξοφλήθηκε» (τελείωσαν και πληρώθηκαν)

Τύπος: Έσοδα = Ώρες Χρέωσης × Τιμή Χρέωσης/Ώρα (χωρίς ΦΠΑ)

Έξοδα

Τα έξοδα υπολογίζονται από:

  • Κόστος Υλικών
  • Κόστος Μετακίνησης (Χιλιόμετρα × Κόστος/km)
  • Κόστος Εργατών (Ώρες × Ωρομίσθιο για κάθε εργάτη)

Τύπος: Έξοδα = Υλικά + Μετακίνηση + Εργάτες

Καθαρά Κέρδη

Τύπος: Κέρδη = Έσοδα - Έξοδα

⚠️ Σημαντικό

Για να είναι σωστά τα στατιστικά, πρέπει να έχεις συμπληρώσει τα κοστολογικά στοιχεία στις εργασίες (υλικά, ώρες, εργάτες)! Αν δεν έχεις βάλει κόστη, τα στατιστικά θα είναι λάθος!

Τι να κάνω αν τα στατιστικά δείχνουν λάθος νούμερα;

Αν τα νούμερα δεν σου βγαίνουν σωστά, έλεγξε τα εξής:

1

Έλεγξε τις καταστάσεις των εργασιών

Οι εργασίες που είναι «Υποψήφιος» ή «Προγραμματισμένη» ΔΕΝ υπολογίζονται στα έσοδα! Μόνο οι «Ολοκληρώθηκε» και «Εξοφλήθηκε» μετράνε!

2

Έλεγξε τα κοστολογικά στοιχεία

Άνοιξε μερικές εργασίες και δες αν έχεις συμπληρώσει:

  • Ώρες Χρέωσης
  • Τιμή Χρέωσης/Ώρα
  • Κόστος Υλικών
  • Εργάτες (αν έχεις)
3

Έλεγξε το φίλτρο έτους

Μήπως έχεις επιλεγμένο λάθος έτος; Δοκίμασε να επιλέξεις «Όλα τα έτη» για να δεις συνολικά νούμερα!

Συμβουλές για τα Στατιστικά

  • 💡 Κοίτα τα στατιστικά κάθε μήνα για να παρακολουθείς την πορεία της δουλειάς σου!
  • 💡 Αν τα κέρδη είναι χαμηλά, κοίτα αν μπορείς να μειώσεις τα έξοδα (π.χ. φτηνότερα υλικά) ή να αυξήσεις την τιμή χρέωσής σου!
  • 💡 Χρησιμοποίησε το γράφημα «Έσοδα ανά Μήνα» για να προβλέψεις πότε θα έχεις περισσότερη ή λιγότερη δουλειά!
  • 💡 Αν βλέπεις ότι χρησιμοποιείς πολύ ένα συγκεκριμένο υλικό, μπορεί να διαπραγματευτείς καλύτερη τιμή με τον προμηθευτή σου!
  • 💡 Ενημέρωνε πάντα την κατάσταση των εργασιών σου για να έχεις ακριβή στατιστικά!

🎯 Στόχος

Στόχος σου είναι να βλέπεις τα καθαρά κέρδη να ανεβαίνουν κάθε μήνα! Αν όχι, κοίτα τι μπορείς να βελτιώσεις: περισσότερες εργασίες, υψηλότερες τιμές, ή μικρότερα έξοδα!
