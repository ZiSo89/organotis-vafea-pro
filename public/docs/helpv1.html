Οδηγός Χρήσης – Οργανωτής Βαφέα

Καλώς ήρθες στον «Οργανωτή Βαφέα». Εδώ θα βρεις έναν απλό οδηγό για όλες τις βασικές λειτουργίες του συστήματος.

Πλοήγηση – Πλαϊνό Μενού

Στο αριστερό μενού θα βρεις τις εξής επιλογές:

  • Αρχική
  • Εργασίες
  • Πελάτες
  • Προσωπικό
  • Αποθήκη (σε ανάπτυξη)
  • Ημερολόγιο
  • Χάρτης
  • Στατιστικά
  • Ρυθμίσεις

Αρχική

Στην αρχική σελίδα βλέπεις μια γρήγορη εικόνα της δουλειάς σου:

  • Πόσες εργασίες υπάρχουν συνολικά
  • Πόσες είναι ενεργές
  • Πόσους πελάτες έχεις
  • Πόσες επισκέψεις/ραντεβού έχεις στο πρόγραμμα

Η σελίδα είναι μόνο για ενημέρωση – δεν χρειάζεται να κάνεις καμία ενέργεια.

Πελάτες

Εδώ μπορείς να προσθέτεις και να διαχειρίζεσαι τους πελάτες σου.

Νέος πελάτης

  • Πάτα το κουμπί «Νέος Πελάτης».
  • Συμπλήρωσε τα βασικά στοιχεία: Όνομα, τηλέφωνο, email, διεύθυνση.
  • Πάτα Αποθήκευση.

Επεξεργασία πελάτη

  • Κάνε κλικ στη γραμμή του πελάτη στη λίστα.
  • Άλλαξε ό,τι χρειάζεται.
  • Πάτα Αποθήκευση.

Διαγραφή πελάτη

  • Επίλεξε τον πελάτη.
  • Πάτα Διαγραφή (αν εμφανίζεται).
  • Επιβεβαίωσε.

Αναζήτηση

Στο πεδίο αναζήτησης γράφεις όνομα ή τηλέφωνο και βρίσκεις εύκολα τον πελάτη.

Εργασίες

Εδώ καταχωρείς όλες τις δουλειές που κάνεις σε πελάτες.

Νέα εργασία

  • Πάτα «Νέα Εργασία».
  • Στη φόρμα υπάρχουν καρτέλες (tabs):
  • Βασικά: Ημερομηνία, πελάτης, κατάσταση.
  • Λεπτομέρειες: Περιγραφή εργασίας, δωμάτια, υλικά.
  • Εργάτες: Ποιοι εργάτες θα πάνε στη δουλειά.
  • Κοστολόγηση: Κόστη, τιμές, τελικά ποσά.
  • Σημειώσεις: Ό,τι έξτρα θες να σημειώσεις.

Όταν είσαι έτοιμος, πάτα Αποθήκευση.

Επεξεργασία εργασίας

  • Κάνε κλικ σε μια εργασία από τη λίστα.
  • Άλλαξε τις πληροφορίες.
  • Πάτα Αποθήκευση.

Αλλαγή κατάστασης

Μπορείς να αλλάζεις την κατάσταση (π.χ. «Σε εξέλιξη», «Ολοκληρωμένη»).

Αναζήτηση & Φίλτρα

Μπορείς να ψάχνεις εργασίες με βάση το όνομα, την ημερομηνία ή την κατάσταση.

Προσωπικό

Εδώ διαχειρίζεσαι τους εργάτες σου.

Νέος εργάτης

  • Πάτα «Νέος Εργάτης».
  • Συμπλήρωσε στοιχεία: Όνομα, ειδικότητα, τηλέφωνο.
  • Αποθήκευση.

Επεξεργασία / Διαγραφή

  • Επίλεξε εργάτη από τη λίστα.
  • Άλλαξε στοιχεία ή πάτα διαγραφή.

Φίλτρα

Μπορείς να φιλτράρεις με βάση ειδικότητα ή κατάσταση (ενεργός/ανενεργός).

Αποθήκη

Η λειτουργία της αποθήκης βρίσκεται σε ανάπτυξη.

Ημερολόγιο

Στο ημερολόγιο βλέπεις τις εργασίες σου με βάση την ημερομηνία.

  • Πάτα μια εργασία για να δεις λεπτομέρειες.
  • Μπορείς να μετακινείς εβδομάδα ή μήνα.
  • Εμφανίζονται αυτόματα ελληνικές αργίες.

Χάρτης

Στον χάρτη βλέπεις:

  • Πού μένουν οι πελάτες σου
  • Πού έχεις προγραμματισμένες εργασίες

Πάτα σε μία καρφίτσα για να δεις τα στοιχεία του πελάτη ή της εργασίας.

Στατιστικά

Εδώ βλέπεις συνολικά στοιχεία για τη δουλειά σου:

  • Έσοδα ανά έτος
  • Πόσες εργασίες έγιναν
  • Ποιες μήνες ήταν πιο δυνατοί

Μπορείς να αλλάξεις έτος από το πάνω δεξιά πεδίο.

Ρυθμίσεις

Εδώ συμπληρώνεις τα στοιχεία της επιχείρησής σου:

  • Όνομα επιχείρησης
  • ΑΦΜ
  • Διεύθυνση
  • Τηλέφωνο

Τα στοιχεία αυτά εμφανίζονται στο σύστημα και θα χρησιμοποιηθούν αργότερα σε προσφορές και τιμολόγια.

Οι σελίδες Προσφορές και Τιμολόγια υπάρχουν στο σύστημα αλλά είναι ακόμα υπό ανάπτυξη.

Τέλος οδηγού

Εάν χρειαστείς οποιαδήποτε βοήθεια, όλες οι σελίδες έχουν απλά κουμπιά και καθαρή εμφάνιση. Μπορείς πάντα να επιστρέψεις σε αυτόν τον οδηγό.
