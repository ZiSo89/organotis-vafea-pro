🚀 Βασικές Λειτουργίες

1️⃣ Αρχική Σελίδα (Dashboard)

  • Επισκόπηση: Συνολικές εργασίες, πελάτες, έσοδα μήνα
  • Επόμενες Επισκέψεις: Προγραμματισμένες εργασίες (κλικ για προβολή)
  • Πρόσφατες Ενέργειες: Τελευταίες αλλαγές
  • Χάρτης: Όλες οι εργασίες με χρωματικό κωδικό:
    • 🔴 Κόκκινο = Σημερινές επισκέψεις
    • 🟢 Πράσινο = Προγραμματισμένες (επόμενες 7 ημέρες)
    • 🔵 Μπλε = Υπόλοιπες εργασίες
  • Dark Mode: Toggle για σκούρο/φωτεινό θέμα

2️⃣ Πελάτες

Προσθήκη Νέου:

  1. Κλικ "➕ Νέος Πελάτης"
  2. Συμπλήρωσε: Ονοματεπώνυμο, Τηλέφωνο, Email
  3. Διεύθυνση: Βάλε πλήρη διεύθυνση (π.χ. "Λεωφόρος Δημοκρατίας 45")
  4. Πόλη: Βάλε την πόλη (π.χ. "Αλεξανδρούπολη")
  5. Κλικ "💾 Αποθήκευση"
✨ Αυτόματες Συντεταγμένες: Η εφαρμογή βρίσκει αυτόματα την τοποθεσία με δωρεάν OpenStreetMap!

Επεξεργασία/Διαγραφή:

  • ✏️ Επεξεργασία: Κλικ στο μολύβι
  • 👁️ Προβολή: Κλικ στο μάτι (εμφανίζει όλα τα στοιχεία + κουμπί Google Maps)
  • 🗑️ Διαγραφή: Κλικ στον κάδο

3️⃣ Εργασίες (Jobs)

Δημιουργία:

  1. Κλικ "➕ Νέα Εργασία"
  2. Επίλεξε Πελάτη από dropdown
  3. Συμπλήρωσε: Περιγραφή, Χρώμα, Κόστος
  4. Επόμενη Επίσκεψη: Κλικ στο πεδίο και επίλεξε ημερομηνία (εμφανίζει ελληνικό ημερολόγιο)
  5. Κατάσταση: Επέλεξε (Προγραμματισμένη, Σε Εξέλιξη, Ολοκληρωμένη, Ακυρωμένη)
  6. Κλικ "💾 Αποθήκευση"

4️⃣ Χάρτες (Maps)

  1. Κλικ "🗺️ Χάρτες" στο sidebar
  2. Βλέπεις Google Maps με όλους τους πελάτες
  3. Κλικ σε Pin: Εμφανίζει όνομα, διεύθυνση, εργασίες
  4. Κουμπί "Οδηγίες": Ανοίγει Google Maps για οδηγίες

💾 Αποθήκευση & Backup

Που Αποθηκεύονται τα Δεδομένα;

Όλα τα δεδομένα αποθηκεύονται τοπικά στον υπολογιστή:

C:\Users\[ΟΝΟΜΑ_ΧΡΗΣΤΗ]\AppData\Local\organotis-vafea\
🔒 Ασφάλεια: Τα δεδομένα δεν φεύγουν ποτέ από τον υπολογιστή σου!

Δημιουργία Backup

  1. Ρυθμίσεις → Εξαγωγή Δεδομένων
  2. Επίλεξε φάκελο (π.χ. Desktop, USB)
  3. Αποθηκεύεται ως painter_backup_YYYYMMDD_HHMMSS.json

Επαναφορά από Backup

  1. Ρυθμίσεις → Εισαγωγή Δεδομένων
  2. Επίλεξε το backup αρχείο (.json)
  3. Επιβεβαίωση - τα δεδομένα αντικαθίστανται
⚠️ Προσοχή: Η εισαγωγή διαγράφει τα τρέχοντα δεδομένα!

🗺️ Πώς Λειτουργούν οι Χάρτες;

Αυτόματη Εύρεση Τοποθεσίας

Όταν προσθέτεις νέο πελάτη:

  1. Βάλε πλήρη διεύθυνση (οδός + αριθμός)
  2. Βάλε πόλη
  3. Η εφαρμογή:
    • 🔍 Ψάχνει στο OpenStreetMap (δωρεάν!)
    • ✅ Βρίσκει συντεταγμένες
    • 📍 Εμφανίζει pin στον χάρτη

Χρωματικός Κωδικός

  • 🔴 Κόκκινο: Επίσκεψη ΣΗΜΕΡΑ
  • 🟢 Πράσινο: Επίσκεψη επόμενες 7 ημέρες
  • 🔵 Μπλε: Όλες οι υπόλοιπες εργασίες

🔧 Αντιμετώπιση Προβλημάτων

Η εφαρμογή δεν ανοίγει

Λύση:

  1. Δεξί κλικ στο .exe → Properties
  2. Unblock (αν υπάρχει στο κάτω μέρος)
  3. Apply → OK
  4. Διπλό κλικ ξανά

Τα δεδομένα χάθηκαν

Λύση: Επαναφορά από backup

Δεν βρίσκει διευθύνσεις

  • Βεβαιώσου ότι η διεύθυνση είναι ελληνική και πλήρης
  • Παράδειγμα: "Λεωφόρος Δημοκρατίας 45, Αλεξανδρούπολη"
  • Το OpenStreetMap χρειάζεται Internet για πρώτη εύρεση
